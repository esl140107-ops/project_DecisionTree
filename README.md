## Проект: Дерево Решений на языке C

Данный проект — это полнофункциональная реализация алгоритма машинного обучения **CART (Classification and Regression Trees)** на чистом языке C (стандарт C99). Фреймворк поддерживает как задачи классификации, так и задачи регрессии, обладая оптимизированным математическим ядром.


## Основные возможности:
1. **Решение задач классификации**: Поддержка бинарной и многоклассовой классификации.
2. **Решение задач регрессии**: Предсказание непрерывных числовых значений.
3. **Алгоритмическая оптимизация (Fast CART)**: Вместо наивного поиска порога за O(N^2) используется предварительная сортировка (Quicksort) и бегущие суммы/гистограммы, что ускоряет поиск сплита до O(N \log N).
4. **Математические критерии (Splitting Criteria)**:
   * **Gini Impurity (Джини)**
   * **Information Gain (Энтропия Шеннона)**
   * **MSE (Дисперсия)**
5. **Важность признаков (Feature Importances)**: Модель автоматически вычисляет и нормализует вклад каждого признака в итоговое решение.
6. **Расширенные метрики качества**:
   * Классификация: Accuracy, Precision, Recall, F1-Score, Матрица ошибок.
   * Регрессия: MSE, RMSE, MAE, R^2 Score (Коэффициент детерминации).
7. **Утилиты для датасетов**: Встроенный парсер CSV, случайное разбиение Train/Test Split.
8. **Визуализация**: Генерация графов в формате `.dot` для Graphviz и текстовый вывод в консоль.

 ## Три контрольных примера, на которых тестируется программа:
iris - проверяется работа с тремя классами одновременно.
diabetes - проверка бинарной классификации.
house_prices - проверка работы алгоритма с числами, а не категориями.

## Инструкция по сборке и запуску

### Требования
* Компилятор GCC или Clang.
* Утилита `make`.
* (Опционально) `graphviz` для рендера графов.
## Инструкция по сборке и запуску

Проект управляется через автоматизированный `Makefile`. Откройте терминал в папке проекта и используйте команды:
### Глобальный запуск (Всё сразу)
```bash
make test_all

Индивидуальные тесты:
make iris — запуск многоклассовой классификации (цветы Ириса).
make diabetes — запуск бинарной классификации (риск диабета).
make house — запуск регрессии (цены на дома).
make clean — очистка проекта от временных файлов и картинок.
